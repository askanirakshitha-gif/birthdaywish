# Birthday Page for Shubham 🎉

A beautiful, personalized birthday page with memories, timeline, countdown, and background music.

## Features
- Custom cursor and animations
- Memory gallery with polaroid-style photos
- Timeline of special moments
- Countdown timer
- Background music toggle
- Responsive design

## Project Structure
```
birthday/
├── index.html          (Main HTML file)
├── images/             (Photo gallery images)
├── audio/              (Background music)
├── vercel.json         (Vercel configuration)
└── README.md           (This file)
```

## How to Add Your Files

### 1. Add Images
Place your image files in the `images/` folder:
- `images/photo1.jpg`
- `images/photo2.jpg`
- `images/photo3.jpg`
- `images/photo4.jpg`
- `images/photo5.jpg`
- `images/photo6.jpg`

### 2. Add Music
Place your audio file in the `audio/` folder:
- `audio/birthday-song.mp3`

## Deployment on Vercel

### Step 1: Initialize Git (if not done)
```bash
cd your-project-folder
git init
git add .
git commit -m "Initial commit"
```

### Step 2: Create a GitHub Repository
1. Go to https://github.com/new
2. Create a new repository (e.g., "birthday-page")
3. Follow GitHub's instructions to push your local repo

### Step 3: Deploy to Vercel
1. Go to https://vercel.com
2. Sign up or log in with GitHub
3. Click "Add New Project"
4. Select your GitHub repository
5. Click "Deploy"
6. Your site will be live in seconds!

### Step 4: Share Your Link
Your Vercel link will be in the format:
`https://your-project-name.vercel.app`

## Customization
- Edit `index.html` to change text, colors, and messages
- Update image paths in the gallery section
- Change the audio file path if needed
- Modify CSS variables for different color schemes
